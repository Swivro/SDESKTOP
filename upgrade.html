How to update sDesktop


If you are on Linux:


Login to your server through SSH with sudo permissions


Navigate to the www directory or public_html directory of your website.


If sDesktop is in a folder called SDESKTOP-master then type the following command outside the SDESKTOP-master directory.


sudo rm -r SDESKTOP-master

Next, run the command below to install the latest version of sDesktop

sudo wget https://github.com/Swivro/SDESKTOP/archive/master.zip && sudo unzip master.zip && rm master.zip


If you are on Windows:

Open File Explorer, navigate to the www or public_html directory of your website, delete everything sDesktop related inside of it including the folder for sDesktop.

Download the sDesktop zip file from the link below and then unzip/extract it inside your www or public_html folder of your website

https://github.com/Swivro/SDESKTOP/archive/master.zip
